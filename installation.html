WELCOME TO THE EMPRETOWN WIKI!
STEP 1)

To get started you will need to install these programs: XAMPP HeidiSQL

STEP 2)

Now you can download the server you can choose between the "main", version which will be updated periodically, or you can download the latest public release. STABLE MAIN

STEP 3)

Download the Linux or Windows artifact files depending on your device (remember to download the latest version). Create a folder inside empiretown with the name "server" and insert the artifacts you just downloaded. Windows Artifact. Linux Artifact.

STEP 4)

Using WinRAR or 7Zip or similar, extract the empiretown folder inside the archive.

STEP 5)

Upload the sql file(database) in HeidiSQL. remember to start XAMPP and click on start MySQL. You will find the database in empiretown/server-data/[SQL]

STEP 6)

Use the "start.bat" file in "server-data" to start the server correctly (if you are on windows platform). Remember that Xampp will need to be started.

STEP 7)

Now just double-click on start.bat to start your server.
